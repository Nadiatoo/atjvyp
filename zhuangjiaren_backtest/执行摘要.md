## 🔧 三、方法论进化（优化后标准）

### 春播判断（恐慌后布局）
**必要条件（满足3项+强势股补跌）:**
- 单日跌幅>2%
- 5日累计跌幅>5%
- 量能放大>1.3倍
- 跌破60日线
- RSI<30
- **新增**: 强势股补跌>15%
- **新增**: 跌停家数>50家（极端恐慌）

### 夏长判断（主升浪持股）
**必要条件:**
- 均线多头排列（5>10>20）
- 收盘价>20日线
- 有明确主线板块
- **新增止损**: 跌破20日线转入冬藏
- **新增止损**: 5日回撤>5%减仓

### 秋收判断（高潮兑现）
**必要条件（满足3项）:**
- 10日涨幅>8%
- 20日涨幅>15%
- 偏离20日线>8%
- RSI>70
- 缩量上涨（量价背离）
- **新增确认**: 板块大块头涨停
- **注意**: 单次高潮不轻易判断为秋收

### 冬藏判断（空仓观望）
**必要条件:**
- 均线空头排列
- 收盘价<60日线
- 无明显恐慌信号
- **退出条件**: 出现恐慌信号→转入春播判断

## ✅ 五、每日分析Checklist

### 盘前准备
- [ ] 查看昨日涨停溢价
- [ ] 查看隔夜外围走势
- [ ] 阅读当日重要新闻

### 季节判断
**春播检查（满足3项+补跌完成）:**
- [ ] 强势股补跌>15%
- [ ] 单日跌幅>2%
- [ ] 5日跌幅>5%
- [ ] 量能放大>1.3倍
- [ ] 跌停家数>50家

**夏长检查:**
- [ ] 均线多头排列
- [ ] 收盘价>20日线
- [ ] 有主线板块
- [ ] ⚠️ 跌破20日线→止损

**秋收检查（满足3项）:**
- [ ] 10日涨幅>8%
- [ ] 偏离20日线>8%
- [ ] RSI>70
- [ ] 板块大块头涨停

**冬藏检查:**
- [ ] 均线空头排列
- [ ] 收盘价<60日线
- [ ] 等待恐慌信号

### 交易计划
**春播:** 分仓布局，单票≤20%，总仓50-70%，止损-3%  
**夏长:** 持有+补涨挖掘，移动止损20日线  
**秋收:** 去弱留强，高潮日减仓30-50%  
**冬藏:** 空仓或≤10%，不抢反弹
