# 飞书权限设置 - 简单方法（不用JSON）

## 🎯 最简单的方法：手动搜索添加

### 步骤1：进入权限管理
```
飞书开放平台 → 你的应用 → 权限管理
```

### 步骤2：点击「批量开通」
在搜索框里逐个搜索下面的权限名，勾选即可。

---

## 📋 需要添加的权限清单

### 云文档相关（搜索关键词：docs）
搜索这些并勾选：
- [ ] `docs:document:write` - 创建和管理文档
- [ ] `docs:document.content:write` - 编辑文档内容  
- [ ] `docs:document.content:read` - 读取文档内容

### 云盘相关（搜索关键词：drive）
- [ ] `drive:drive:write` - 访问和管理你的云空间
- [ ] `drive:file:write` - 上传和下载文件
- [ ] `drive:folder:write` - 创建和管理文件夹

### 知识库相关（搜索关键词：wiki）
- [ ] `wiki:wiki:write` - 创建和管理知识库
- [ ] `wiki:node:write` - 管理知识库节点

### 消息相关（已有，但确认一下）
- [ ] `im:message:send_as_bot` - 以机器人身份发送消息

---

## 🔍 搜索技巧

在「批量开通」页面的搜索框：
1. 输入 `docs` → 显示所有云文档权限 → 勾选前3个
2. 输入 `drive` → 显示所有云盘权限 → 勾选前3个  
3. 输入 `wiki` → 显示所有知识库权限 → 勾选前2个

然后点击「确定」即可。

---

## ❓ 如果还是找不到

### 方案A：直接点「申请更多权限」
1. 在权限管理页面
2. 点击「申请更多权限」按钮
3. 在分类中找到：
   - **云文档** → 全选
   - **云盘** → 全选
   - **知识库** → 全选

### 方案B：使用API直接授权（技术方案）
如果你懂技术，可以用接口直接授权：

```bash
# 需要你的App Access Token
curl -X POST \
  'https://open.feishu.cn/open-apis/application/v6/tenant/app_permissions' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "permissions": [
      "docs:document:write",
      "docs:document.content:write",
      "drive:drive:write",
      "drive:file:write",
      "wiki:wiki:write"
    ]
  }'
```

---

## ✅ 最笨但最保险的方法

如果以上都不行：

1. 在权限管理页面
2. 不要点「批量开通」
3. 直接看页面上已有的权限列表
4. 找到「云文档」「云盘」「知识库」分类
5. 逐个点击「申请」按钮

每个权限右边应该有个「申请」或「开通」按钮，点一下就行。
