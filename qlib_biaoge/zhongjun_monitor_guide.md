# 中军健康度监控清单 - 真夏长识别关键

## 一、中军定义与筛选

### 什么是中军？
- **市值**：流通市值 100-500亿（太小拉不动，太大涨不动）
- **成交**：日均成交 >10亿（大资金战场）
- **地位**：板块内业务最正宗、机构持仓最多
- **走势**：趋势稳健，波动小于龙头

### 中军 vs 龙头
| 维度 | 龙头 | 中军 |
|:---|:---|:---|
| 市值 | 30-100亿 | 100-500亿 |
| 涨幅 | 连板爆发 | 趋势上涨 |
| 资金 | 游资主导 | 机构+游资 |
| 作用 | 情绪载体 | 资金载体 |
| 风险 | A字杀 | 阴跌 |

## 二、中军健康度六大指标

### 1. 涨跌幅健康度 (权重25%)
```
✅ 优秀：中军平均涨幅 >5%
🟡 良好：中军平均涨幅 3-5%
🟠 一般：中军平均涨幅 0-3%
🔴 危险：中军平均涨幅 <0%
```

**关键观察**：
- 龙头涨停时，中军是否跟涨？
- 板块上涨时，中军是否创新高？
- 中军集体负收益 = 板块退潮信号

### 2. 杀跌收回速度 (权重25%)
```
✅ 优秀：1天内收回跌幅
🟡 良好：2天内收回
🟠 一般：3-5天收回
🔴 危险：>5天或无法收回
```

**关键观察**：
- 日内跳水后能否V回？
- 次日低开能否快速翻红？
- 收回速度 = 资金信心指标

### 3. 成交活跃度 (权重20%)
```
✅ 优秀：日均成交 >20亿
🟡 良好：日均成交 10-20亿
🟠 一般：日均成交 5-10亿
🔴 危险：日均成交 <5亿
```

**关键观察**：
- 大资金能否自由进出？
- 缩量上涨 = 隐患
- 放量滞涨 = 出货

### 4. 亏钱效应 (权重20%)
```
✅ 优秀：跌幅>5%的中军 <10%
🟡 良好：跌幅>5%的中军 10-20%
🟠 一般：跌幅>5%的中军 20-30%
🔴 危险：跌幅>5%的中军 >30%
```

**关键观察**：
- 中军是否批量下跌？
- 有无中军跌停？
- 亏钱效应扩散 = 撤退信号

### 5. 技术形态 (权重10%)
```
✅ 优秀：沿5日线上行，均线多头排列
🟡 良好：沿10日线上行，5日/10日金叉
🟠 一般：20日线附近震荡
🔴 危险：跌破20日线，均线空头排列
```

## 七、与真夏长评分系统的整合

中军健康度（20%权重）+ 其他维度（80%权重）= 真夏长总分

**关键逻辑**：
- 中军健康度 ≥15分：加分项，真夏长概率高
- 中军健康度 <10分：即使其他维度好，也要警惕假夏长
- 中军健康度 =0分：直接判定假夏长，无论其他维度

**实战原则**：
> 中军是板块的灵魂，中军不健康，龙头再强也是独舞。
> 真夏长必须是龙头+中军+跟风的三位一体。
