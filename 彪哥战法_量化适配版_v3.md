# 彪哥战法 · 量化适配版（v3.0）

> 基于2021-2025年回测优化（胜率61.9%，冬藏判断准确率93.78%）  
> **核心升级**：全面适配量化交易时代特征

---

## 一、量化时代核心认知

### 1.1 量化交易已成为市场主导力量

**规模与占比**
- 2021年：量化规模突破1万亿，占比约15%
- 2024年：量化规模约1.5万亿，占比约25%
- 预计2025年：占比将达到30%

**影响范围**
- 几乎所有活跃股都有量化参与
- 连板股90%以上有量化席位
- 龙虎榜量化席位占比超过40%

### 1.2 量化改变的市场规律

| 传统规律 | 量化时代变化 | 对战法影响 |
|----------|--------------|------------|
| 连续涨停打开空间 | 连板高度受限（5板天花板） | 降低连板预期 |
| V型反转确认底部 | 底部反复震荡（锯齿形） | 延长建仓周期 |
| 量价齐升趋势明确 | 阶梯式上涨，日内波动大 | 适应做T策略 |
| 放量滞涨是顶 | 对倒出货维持高位 | 提前兑现 |
| 情绪冰点抄底 | 量化多杀多，闪崩增多 | 严格空仓纪律 |

### 1.3 量化识别的重要性

**为什么必须识别量化？**
1. 量化是对手盘，了解对手才能获胜
2. 量化行为可预测（算法趋同）
3. 利用量化规律可提高胜率
4. 避免被量化收割（假突破、诱多）

## 二、量化识别指标体系（新增）

### 2.1 盘口识别法

**大单夹小单（量化控盘）**
```
特征：
- 买卖盘出现明显"夹心"结构
- 大单与大小单之间有断层
- 盘口厚度不均匀

判断：
✅ 买入信号：托底大单稳，压盘大单轻
❌ 卖出信号：托底大单撤，压盘大单重
```

**对倒盘（量化建仓/出货）**
```
特征：
- 同一时间买卖大单对敲
- 成交量突增但价格变动小
- 龙虎榜量化席位买卖同现

判断：
✅ 建仓信号：低位对倒，后续上涨
❌ 出货信号：高位对倒，后续下跌
```

### 2.2 分时识别法

**锯齿形（量化T0）**
```
特征：
- 价格反复上下，无明显方向
- 振幅较小（<3%），频率高
- 持续时间长（>30分钟）

应对：
- 不参与震荡，等方向明确
- 低吸高抛做T机会
- 放量突破跟进
```

**织布机（量化控制）**
```
特征：
- 价格在极窄区间震荡（<1%）
- 成交量萎缩
- 买卖盘挂单密集

应对：
- 回避观望，无参与价值
- 等待放量突破
- 不预测方向
```

**冲击波（量化诱多/诱空）**
```
特征：
- 快速拉升/杀跌后迅速收回
- 成交量脉冲式放大
- 无板块共振

应对：
- 不追突破，防假突破
- 不恐慌割肉，防假跌破
- 看板块共振确认
```

### 2.3 量能识别法

**间歇性放量（量化分批操作）**
```
特征：
- 量能脉冲式释放，非持续
- 放量日与缩量日交替
- 与价格走势背离

判断：
建仓期：低位间歇放量，逐步上涨
出货期：高位间歇放量，滞涨下跌
```

**异常缩量（量化锁仓）**
```
特征：
- 成交量突然萎缩至前期1/3
- 价格波动收窄
- 持续3-5日

判断：
✅ 上涨中继：缩量整理后继续涨
❌ 顶部信号：缩量滞涨后下跌
```

### 2.4 龙虎榜识别法

**常见量化席位清单**
```
第一梯队（头部量化）：
- 华泰证券股份有限公司总部
- 中国国际金融股份有限公司上海分公司
- 中信证券股份有限公司上海分公司

第二梯队（中型量化）：
- 国泰君安证券股份有限公司总部
- 招商证券股份有限公司深圳深南东路营业部
- 东方财富证券股份有限公司拉萨团结路
```

**量化席位行为解读**
```
买入为主：
- 量化建仓，可关注
- 但需防次日砸盘

卖出为主：
- 量化出货，回避
- 如有机构接盘，可观察

买卖同现：
- 量化做T，震荡为主
- 不追涨杀跌
```

**机构+量化混合信号**
```
买入：机构专用 + 量化席位
卖出：量化席位主导

解读：
- 机构建仓，量化做T
- 可跟随机构，但需耐心
- 量化砸盘时是低吸机会
```

## 三、四季战法量化适配（核心升级）

### 3.1 春播期：量化吸筹识别

**传统春播标准（v2.0）**
- 强势股补跌≥10%
- 跌停家数>50家
- 成交量>1.5倍5日均量
- 次日有板块引领反弹

**量化适配升级（v3.0）**
```
新增识别指标：
1. 量化吸筹特征（满足≥1项）
   - 分时呈锯齿形（底部震荡>3日）
   - 量能间歇性放量
   - 龙虎榜量化席位低位买入

2. 量化锁仓信号
   - 成交量萎缩至阶段低点
   - 盘口出现大单托底
   - 抛压减轻（大单夹小单）

3. 量化启动前兆
   - 底部放量突破（量>2倍）
   - 量化席位加仓
   - 板块内多股同步启动
```

**春播建仓策略（量化适配）**
```
仓位管理：≤30%（维持不变）

建仓节奏：
Day1: 确认春播信号，建仓10%
Day2-3: 观察量化吸筹特征，如确认再加10%
Day4-5: 放量突破确认，加满至30%

止损纪律：
- 跌破春播低点-7%，止损
- 量化席位连续3日净卖出，止损
- 板块跌破20日线，止损
```

### 3.2 夏长期：量化拉升适应

**传统夏长标准（v2.0）**
- 主线板块连续3日领涨
- 涨停家数>100家
- 连板高度≥4板
- 赚钱效应持续3日+

**量化适配升级（v3.0）**
```
核心调整：降低连板预期，适应阶梯式上涨

新标准：
1. 阶梯式上涨确认（满足≥2项）
   - 涨一天、调一天，重心上移
   - 连板高度≥3板（降低 from 4板）
   - 5日内涨幅>20%
   - 量化席位净买入

2. 量化做T特征（正常现象）
   - 日内波动>8%（振幅加大）
   - 分时锯齿形（非出货）
   - 量能维持，未异常放大

3. 夏长延续信号
   - 量化席位持续净买入
   - 龙虎榜机构+量化混合
   - 板块阵型完整（先锋+中军+后排）
```

**夏长持仓策略（量化适配）**
```
仓位管理：≤70%（维持不变）

持股策略：
- 核心仓位持有（不受日内波动影响）
- 利用量化做T，日内高抛低吸（可选）
- 重点关注量化席位动向

补涨挖掘：
- 量化持仓低位的同属性股
- 锯齿形吸筹阶段的个股
- 量化席位新进个股

量化撤退信号（减仓）：
- 量化席位连续2日净卖出
- 高位对倒盘出现
- 连板高度无法突破5板
```

### 3.3 秋收期：量化出货识别

**传统秋收标准（v2.0）**
- 板块指数单日涨幅>7%
- 千亿市值个股涨停
- 跟风分支鸡犬升天
- 连板高度达到近30日峰值

**量化适配升级（v3.0）**
```
新增识别指标：
1. 量化出货特征（满足≥1项）
   - 高位对倒盘（大单买卖对敲）
   - 龙虎榜量化席位大额净卖出
   - 间歇性放量滞涨
   - 假突破后快速回落

2. 对倒出货识别
   - 成交量放大但价格不涨
   - 盘口大单夹小单，维持价格
   - 龙虎榜买卖金额接近（对敲）

3. 量化撤退信号
   - 量化席位连续3日净卖出
   - 高位锯齿形（出货而非做T）
   - 板块内多股同步量化出货
```

**秋收兑现策略（量化适配）**
```
仓位管理：逐步降至≤20%

兑现节奏：
第一档（高潮日）：
- 清仓跟风补涨股
- 减仓量化席位大额卖出的个股
- 保留量化席位仍净买入的核心股

第二档（量化出货确认）：
- 对倒盘出现时，再减50%
- 量化连续2日净卖出，清仓该个股
- 保留机构主导的个股

第三档（趋势破位）：
- 跌破10日线，清仓
- 量化席位全面撤退，彻底离场
- 不等待反包（量化时代反包减少）
```

### 3.4 冬藏期：量化多杀多防范

**传统冬藏标准（v2.0）**
- 高位股持续杀跌
- 题材半日游
- 涨停家数<50家
- 炸板率>40%

**量化适配升级（v3.0）**
```
新增识别指标：
1. 量化多杀多特征（满足≥1项）
   - 无利空闪崩（量化踩踏）
   - 流动性危机（大单砸盘无承接）
   - 龙虎榜量化席位集体卖出
   - 日内跌幅>15%（量化平仓）

2. 量化踩踏识别
   - 多只高位股同步跌停
   - 量化席位大额净卖出（>1亿）
   - 盘口出现多档大单砸盘

3. 量化抄底信号（冬藏转春播）
   - 量化席位开始小额净买入
   - 分时锯齿形吸筹
   - 成交量萎缩至地量
```

**冬藏空仓策略（量化适配）**
```
仓位管理：≤10%（维持不变）

空仓纪律：
- 量化席位集体卖出期间，绝对空仓
- 多杀多阶段，不抢反弹
- 等待量化吸筹信号出现

观察重点：
- 量化席位何时转向
- 强势股补跌是否≥10%
- 是否出现量化建仓信号

春播准备：
- 量化席位连续2日净买入
- 分时锯齿形吸筹
- 板块共振确认
```

## 四、量化时代主线识别升级

### 4.1 量化参与的主线特征

**特征1：量化席位密集介入**
```
正向信号：
- 龙头股的龙虎榜有量化席位
- 量化席位净买入为主
- 机构+量化混合（最佳）

负向信号：
- 无量化席位参与（非主线）
- 量化席位净卖出（出货期）
- 纯散户炒作（无持续性）
```

**特征2：分时走势一致性**
```
正向信号：
- 板块内多股同步锯齿形吸筹
- 阶梯式上涨节奏一致
- 量化做T痕迹明显

负向信号：
- 各股走势分化严重
- 无明显的量化控盘痕迹
- 纯情绪推动（无基本面）
```

**特征3：量能特征一致性**
```
正向信号：
- 板块内多股间歇性放量
- 缩量整理时同步缩量
- 放量突破时同步放量

负向信号：
- 量能分布不均匀
- 无明显的量化建仓痕迹
- 放量滞涨（出货）
```

### 4.2 量化时代的板块阵型

**先锋（连板龙头）**
```
量化时代特征：
- 连板高度受限（3-5板）
- 断板后走趋势（非A杀）
- 量化席位频繁做T

识别要点：
- 龙虎榜量化席位净买入
- 分时锯齿形（非出货）
- 断板后仍能维持高位
```

**中军（大市值逻辑核心）**
```
量化时代特征：
- 机构+量化混合主导
- 阶梯式上涨为主
- 日内波动大，适合波段

识别要点：
- 机构席位大额买入
- 量化席位做T痕迹
- 均线多头排列
```

**后排（低位补涨）**
```
量化时代特征：
- 量化挖掘低位股
- 补涨速度快（1-2日）
- 出货也快（不恋战）

识别要点：
- 量化席位新进
- 低位锯齿形吸筹
- 快速补涨后快速出货
```

### 4.3 量化参与的买卖点优化

**买点优化**
```
最佳买点：
1. 量化吸筹确认（分时锯齿形+间歇放量）
2. 量化席位净买入首日
3. 板块共振启动日

次佳买点：
1. 量化做T回调（日内低点）
2. 阶梯式上涨的调整日
3. 量化席位加仓日

避免的买点：
1. 量化对倒盘（诱多）
2. 量化席位大额净卖出日
3. 假突破当日
```

**卖点优化**
```
必须卖点：
1. 量化席位连续2日净卖出
2. 高位对倒盘出现
3. 假突破后回落

考虑卖点：
1. 量化做T日内高点
2. 阶梯式上涨的上涨日（高抛低吸）
3. 板块高潮日（减仓）

避免的卖点：
1. 量化吸筹期（锯齿形）
2. 量化正常做T（非出货）
3. 板块正常调整
```

## 五、量化时代的仓位与风控

### 5.1 仓位管理升级

| 季节 | 仓位上限 | 量化调整 |
|------|----------|----------|
| 春播 | ≤30% | 量化吸筹确认后可加至35% |
| 夏长 | ≤70% | 量化持续净买入可保持 |
| 秋收 | ≤20% | 量化出货信号出现立即降至10% |
| 冬藏 | ≤10% | 量化多杀多时绝对空仓 |

### 5.2 止损纪律升级

**传统止损（v2.0）**
- 个股跌破买入价-7%，止损
- 板块跌破20日线，清仓

**量化适配止损（v3.0）**
```
新增止损信号：
1. 量化席位连续2日净卖出，止损
2. 高位对倒盘出现，止损
3. 无利空闪崩（量化踩踏），止损

放宽止损（量化做T阶段）：
- 日内波动大，允许-10%再止损
- 但量化出货信号必须立即止损
```

### 5.3 止盈策略升级

**量化时代止盈原则**
```
核心原则：见好就收，不追求最后一个铜板

止盈节奏：
- 达到目标收益（10-20%），减仓50%
- 量化出货信号出现，清仓
- 板块高潮日，清仓补涨股

动态止盈：
- 浮盈>15%后，回撤-5%止盈
- 浮盈>30%后，回撤-10%止盈
- 量化席位转向即止盈
```

## 七、心法口诀（量化版）

### 春播
- 无恐慌不开仓，无量能不开仓
- 强势股补跌≥10%，才是真春播
- **量化吸筹锯齿形，底部震荡要耐心**
- 分仓布局≤30%，不怕倒春寒

### 夏长
- 持股待涨≤70%，让利润奔跑
- **阶梯上涨是常态，连板高度五板顶**
- 量化做T波动大，高抛低吸可参与
- 高潮信号≥2项，启动秋收

### 秋收
- 去弱留强，逐步兑现至≤20%
- **对倒出货要识别，假突破是陷阱**
- 量化席位转向日，必须清仓
- 不追求卖在最高点，吃完鱼身即可

### 冬藏
- 管住手≤10%，脉冲是诱饵
- **量化多杀多凶险，严格空仓保命**
- 观察量化何时转向，等待真春播
- 复利的前提是回避大回撤
