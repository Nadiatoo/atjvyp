## 🔍 二、监管风险前置指标（替代原"监管信号"）

### 核心认知调整

**原方案（错误）**：
- 把"监管强度"作为评分指标（直接打分）

**新方案（正确）**：
- 监管是**结果**，不是**原因**
- 要找**导致监管的前置因子**
- 建立**监管风险预警系统**（红灯/黄灯/绿灯）

### 监管触发的前置因子

```
┌─────────────────────────────────────────────────────────────┐
│                  监管风险预警指标体系                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🔴 红灯警告（监管即将出手，立即减仓）                        │
│  ├── 连板高度>10板 且 连续3日加速                           │
│  ├── 20cm连板>5板                                           │
│  ├── 个股10日内涨幅>150%                                     │
│  ├── 板块内多股同时触发100%异动（批量监管）                  │
│  └── 妖股出现"天地板"后次日反包（极端情绪）                  │
│                                                             │
│  🟡 黄灯预警（监管可能关注，控制仓位）                        │
│  ├── 连板高度7-10板                                         │
│  ├── 20cm连板3-4板                                          │
│  ├── 个股7日涨幅>100%                                        │
│  ├── 板块成交额占全市场>25%（过度拥挤）                      │
│  └── 出现"特停"个案（警示作用）                              │
│                                                             │
│  🟢 绿灯通行（监管无动作，正常操作）                          │
│  ├── 连板高度<7板                                           │
│  ├── 板块成交额占比<15%                                      │
│  └── 无个股触发严重异常波动                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 监管应对策略

| 预警级别 | 信号 | 应对策略 |
|:---:|:---|:---|
| **🔴 红灯** | 触发3个以上红灯条件 | 清仓或减仓至20%以下，回避监管风险 |
| **🟡 黄灯** | 触发2个以上黄灯条件 | 减仓至50%，去弱留强，只做龙头 |
| **🟢 绿灯** | 无预警 | 正常操作，按四季标准执行 |

### 历史案例验证

**案例1：2026年1月商业航天退潮**
- 胜通14板（🔴红灯）
- 批量100%异动（🔴红灯）
- 村里组合拳降温（结果）
- **应对**：12日减仓，20日离场 ✅

**案例2：2025年人形机器人**
- 连板高度控制在7板以内（🟢绿灯）
- 趋势为主，连板为辅（规避监管）
- **应对**：5日线持有，未触发监管 ✅

**案例3：2025年创新药**
- 以趋势为主，少有连板（🟢绿灯）
- 机构主导，游资参与少（天然规避监管）
- **应对**：按趋势策略执行 ✅

## 💡 四、关键认知更新

### 1. 监管是结果，不是原因
- 要找导致监管的前置因子（连板高度、涨幅、异动数量）
- 建立红绿灯预警系统，提前规避
- 不同题材规避监管方式不同（商业航天连板被压，人形机器人改趋势）

### 2. 产业阶段是核心
- 商业化前夜（订单即将放量）= 最佳介入时机
- 产品化落地期 = 仍可参与，但性价比下降
- 概念验证期 = 高风险，纯博弈

### 3. 催化密度决定节奏
- 高频催化（月度）= 持续性强，适合趋势持有
- 低频催化（年度）= 脉冲行情，适合事件驱动
- 商业航天：季度大事件（可回收火箭）
- 人形机器人：月度视频/订单更新
- 创新药：季度临床数据/管线进展

### 4. 不同题材不同的钱
- 商业航天：政策驱动的钱（顶层设计→地方配套→订单落地）
- 人形机器人：产业趋势的钱（海外映射→国内跟随→业绩兑现）
- 创新药：政策拐点的钱（医保谈判缓和→估值修复→业绩释放）

### 5. 监管适应能力
- 商业航天：村里降温后→大二波需等可回收成功（新催化）
- 人形机器人：主动规避连板→趋势抱团+蛇形走位
- 创新药：机构主导→天然规避监管，但依赖宏观
