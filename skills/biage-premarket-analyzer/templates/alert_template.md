# 板块阵型实时提醒模板

## 模板说明
本模板用于每日盘前/盘中板块阵型分析，自动填充扫描结果

---

## 标准输出模板

```
【日期】{date}
【季节判断】{season}
【主线板块】{main_sector}

【板块阵型分析】

◆ 龙头候选（按潜力排序）:
{dragon_list}

◆ 中军候选:
{general_list}

【阵型健康度】
评分：{formation_score}/100
- 先锋：{dragon_check}（{dragon_status}）
- 中军：{general_check}（{general_status}）
- 后排：{follower_check}（{follower_status}）
- 整体评价：{formation_type}

【操作建议】
{suggestions}

【风险提示】
{risk_warning}
```

## 龙头候选填充模板

```
  {rank}. {name}（{symbol}）
     - 市值：{market_cap}亿（流通{circulating_cap}亿）
     - 连板数：{lianban}板
     - 涨停时间：{first_time}
     - 换手率：{turnover}%
     - 龙虎榜：{lhb_status}
     - 龙头潜力：{potential}（{reasons}）
```

### 龙头潜力判定
- **高**: 评分>=70分，具备成为龙头条件
- **中**: 评分50-69分，需观察确认
- **低**: 评分<50分，跟风概率大

---

## 中军候选填充模板

```
  {rank}. {name}（{symbol}）
     - 市值：{market_cap}亿
     - 涨幅：{change}%
     - 龙虎榜：{lhb_status}
     - 中军地位：{status}（{reasons}）
```

### 中军地位判定
- **核心**: 评分>=60分，板块中军
- **跟风**: 评分<60分，准中军或假中军

---

## 阵型健康度判定

### 评分标准
| 评分 | 阵型类型 | 策略 |
|------|----------|------|
| 80-100 | 进攻型 | 积极参与 |
| 50-79 | 防守型 | 控制仓位 |
| <50 | 松散型 | 观望为主 |

### 阵型要素检查
- **先锋**: 是否有3板以上连板龙头
- **中军**: 是否有千亿级中军坐镇
- **后排**: 是否有5家以上补涨股

---

## 操作建议填充

### 进攻型策略
```
• 阵型完整，可积极参与
• 关注龙头分歧买点
• 中军可作为稳健配置
• 后排补涨谨慎追高
```

### 防守型策略
```
• 阵型尚可，控制仓位参与
• 等待龙头确认强度
• 中军打底仓观察
• 避免后排跟风股
```

### 松散型策略
```
• 阵型松散，观望为主
• 等待龙头走出来
• 不急于建仓
• 关注板块持续性
```
