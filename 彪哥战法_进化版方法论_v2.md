# 彪哥战法 · 进化版方法论（v2.0）

> 原"庄稼人战法"，现统一更名为"彪哥战法"以便记忆

> 基于2021-2025五年回测优化（胜率61.9%，冬藏判断准确率93.78%）

---

## 一、核心进化点（vs 原版）

| 维度 | 原版 | 进化版（基于回测） |
|------|------|-------------------|
| **春播判断** | 恐慌杀跌即开仓 | 增加量能验证（>1.5倍均量）+ 强势股补跌≥10% |
| **周期长度** | 固定四季轮回 | 量化时代周期缩短，春播窗口收窄 |
| **判断层级** | 大盘四季 | 增加板块级四季判断（应对结构性行情） |
| **仓位管理** | 无明确标准 | 冬藏≤10%、春播≤30%、夏长≤70%、秋收≤20% |
| **结构性行情** | 未特别处理 | 指数与板块背离时，以板块为准 |

## 二、每日市场分析三步法（进化版）

### Step 1：判断当前季节（大盘+板块双维度）

**大盘季节判断标准：**

| 季节 | 判断标准（满足≥2项） | 仓位建议 |
|------|---------------------|----------|
| **春播** | ①强势股补跌≥10%完成<br>②跌停家数>50家（恐慌）<br>③成交量>1.5倍5日均量<br>④次日有板块引领反弹 | ≤30% |
| **夏长** | ①主线板块与大盘共振向上<br>②涨停家数>100家<br>③连板高度≥4板<br>④赚钱效应持续3日+ | ≤70% |
| **秋收** | ①板块全面高潮（大块头涨停）<br>②跟风分支鸡犬升天<br>③连板高度达到近期峰值<br>④涨停家数/炸板率>5:1 | ≤20% |
| **冬藏** | ①高位股持续杀跌<br>②题材半日游<br>③涨停家数<50家<br>④炸板率>40% | ≤10% |

**板块季节判断（结构性行情专用）：**
- 大盘冬藏期，某板块若满足春播三要素，可独立春播
- 大盘夏长期，某板块若已高潮，可独立秋收

---

### Step 2：识别主线板块（三要素+量化验证）

**主线三要素检查表：**

**☑️ 天时（情绪周期）**
- [ ] 板块处于春播或夏长初期
- [ ] 能带动市场情绪共振（涨停家数行业占比>20%）

**☑️ 地利（位置与筹码）**
- [ ] 板块指数近20日涨幅排名后50%（低位）
- [ ] 板块内个股平均偏离20日均线<10%
- [ ] 近期无大量获利盘（换手率<5%）

**☑️ 人和（产业逻辑）**
- [ ] 有清晰的产业预期/政策催化
- [ ] 相比其他板块逻辑鹤立鸡群
- [ ] 新闻热度环比增加（搜索指数上升）

**主线识别的四个特征（尽早锁定）：**

**特征1：与大盘共振反弹**
- 春播日能有效带动大盘共振向上
- 板块涨幅与大盘涨幅同步，非独自上涨
- 涨停家数行业占比>20%
- ⚠️ 误区：独自上涨无共振的不是主线

**特征2：倒春寒中表现韧性**
- 大盘回调时，主线板块跌幅小于大盘
- 板块内核心个股抗跌，甚至逆势上涨
- 洗盘后能快速收复失地

**特征3：天时地利人和兼备**
- 天时：春播期共振向上
- 地利：位置低（20日涨幅后50%），无获利盘抛压
- 人和：产业逻辑清晰，有预期差
- 三者兼备，更容易持续得到多方力量共振

**特征4：板块阵型结构合理（进攻阵型）**
```
        先锋（连板龙头）
           ↓
      中军（逻辑最纯、体量最大）
           ↓
    后排补涨（低位跟随）
```

- **先锋**：连板人气股，小市值，打开空间，聚集目光
- **中军**：逻辑最纯正的大市值公司，大资金主战场，稳定军心
- **后排**：补涨个股，阶梯式跟随

**阵型良好的标志：**
- ✅ 先锋先领涨打开空间
- ✅ 中军稳步推进（非一日游）
- ✅ 后排快速补涨（梯队完整）
- ❌ 无阵型的板块：一窝蜂一天高潮，无纵深，一跌就散

**误区警示：**
- **误区1**：春播日第一个反弹的就是主线？❌ 未必，要看是否共振
- **误区2**：第一个共振反弹的就是主线？❌ 未必，要看是否有地利人和
- **正解**：共振+倒春寒韧性+地利人和+阵型合理=主线

**结构性行情特殊处理：**
- 若指数与板块背离（指数跌板块涨），以板块为准
- 典型案例：2023年AI行情，指数震荡但AI主线持续

### Step 3：制定交易计划（含仓位与风控）

**春播期计划模板：**
```
【季节判断】春播期（大盘/板块）
【判断依据】
- 强势股补跌：XX板块补跌XX%（≥10%确认）
- 恐慌信号：跌停XX家，成交量XX亿（>1.5倍均量）
- 反弹引领：XX板块于XX日引领反弹

【主线候选】
1. XX板块：天时✓地利✓人和✓（理由：XXX）
2. XX板块：天时✓地利✓（理由：XXX）

【交易计划】
- 总仓位：≤30%
- 分仓：每个板块≤10%，共2-3个板块
- 止损：板块跌破20日线/个股跌破买入价-7%
- 预期：持有时长5-10日，目标收益10-20%

【风险提示】
- 假春播信号：若5日内未出现主线引领，减仓至10%
- 倒春寒应对：若遇大盘回调，补票上车（加仓≤10%）
```

**夏长期计划模板：**
```
【季节判断】夏长期（大盘/板块）
【主线确认】XX板块已引领大盘/板块共振3日+

【持仓策略】
- 核心仓位：≤50%（主线龙头）
- 补涨仓位：≤20%（低位+主线属性）
- 每日检查：是否出现秋收信号

【补涨挖掘】
- 方向：低位的XX属性股（<20日均线）
- 标签：次新/低价/小市值/最新事件
- 卖点：主线高潮日（批量涨停）必须清仓补涨

【秋收预警】
- 信号1：板块指数单日大涨7%+
- 信号2：大块头（千亿市值）涨停
- 信号3：跟风分支鸡犬升天
- 应对：启动秋收，去弱留强
```

**秋收期计划模板：**
```
【季节判断】秋收期（高潮一致）
【高潮确认】
- 板块涨幅：XX板块今日+XX%
- 连板高度：最高XX板（近期峰值）
- 涨停结构：大块头涨停XX家，跟风XX家

【兑现节奏】
第一档（高潮日）：
- 清仓所有跟风补涨股（减仓XX%）
- 减仓中后排个股（减仓XX%）

第二档（反包后）：
- 主线反包技能卡用完后
- 清仓中后排，保留核心龙头≤20%

第三档（次反包/破位）：
- 次反包技能卡用完或跌破10日线
- 彻底清仓，进入冬藏

【底线纪律】
- 无论后续涨跌，跌破20日线无条件清仓
- 不追求卖在最高点，吃完鱼身即可
```

## 三、四季转换识别信号（量化版）

### 冬藏 → 春播

**必备条件（全部满足）：**
1. 近期所有逆势逞强的板块都补跌≥10%
2. 跌停家数>50家（或>5%个股跌停）
3. 成交量>1.5倍5日均量（恐慌放量）
4. 次日有板块站出来引领反弹（涨停家数行业占比>20%）

**易错点（假春播识别）：**
- 冬藏期会有多次反弹，只有强势股补跌后的才是真春播
- 2021年抱团瓦解期，多次假春播导致套牢

### 春播 → 夏长

**确认信号（满足≥2项）：**
1. 主线板块连续3日领涨（涨幅排名前10）
2. 板块内出现4连板以上龙头
3. 补涨分支开始活跃（跟风涨停>10家）
4. 大盘与板块共振向上（同涨同跌）

### 夏长 → 秋收

**高潮一致信号（满足≥2项）：**
1. 板块指数单日涨幅>7%
2. 板块内千亿市值个股涨停或冲击涨停
3. 跟风分支鸡犬升天（无关个股也被炒作）
4. 连板高度达到近30日峰值
5. 涨停家数/炸板率>5:1（情绪过热）

### 秋收 → 冬藏

**确认信号（满足≥2项）：**
1. 主线板块反包技能卡用完（通常1-2次反包）
2. 板块核心股见顶回落（跌破5日线）
3. 题材持续性变差（半日游，早盘涨停午后炸板）
4. 涨停家数<50家，炸板率>40%
5. 高位股开始A字杀（连续跌停）

## 四、不同市道的周期特征（基于回测）

### 大盘上涨趋势中（2020-2021上半年）
- **冬藏期**：1-2天，快速洗牌
- **夏长期**：很长，多轮情绪周期叠加
- **策略**：重仓位参与（夏长≤70%），每一次春播都是机会
- **胜率**：春播58.82%，冬藏93.78%

### 大盘震荡市中（2021下半年-2022）
- **四季时长**：相对均衡，周期缩短
- **策略**：控制仓位，快进快出，不恋战
- **关键**：选对主线比择时更重要（结构性行情）

### 大盘下跌趋势中（2022-2024年初）
- **冬藏期**：很长，多次冰点（2022年有5次假春播）
- **夏长期**：很短，闪电战（3-5日）
- **策略**：降低预期，小仓位博弈（春播≤20%），见好就收
- **纪律**：等强势股补跌10%+多方持续性确认后再入场

### 量化时代特征（2021年后）
- **情绪周期缩短**：春播窗口从3-5日收窄至1-2日
- **反包技能卡弱化**：连板高度受限，反包次数减少
- **结构性分化**：指数与板块背离成常态，需增加板块级判断

## 六、心法口诀（进化版）

**春播：**
- 无恐慌不开仓，无量能不开仓（>1.5倍）
- 强势股补跌≥10%，才是真春播
- 分仓布局≤30%，不怕倒春寒

**夏长：**
- 持股待涨≤70%，让利润奔跑
- 补涨看位置（<20日线），涨起来就没性价比
- 高潮信号≥2项，启动秋收

**秋收：**
- 去弱留强，逐步兑现至≤20%
- 反包技能卡用完后彻底离场
- 没有完美交易，赚了就跑，不追求最后一个铜板

**冬藏：**
- 管住手≤10%，脉冲是诱饵
- 观察强势股补跌10%+，等待真春播
- 复利的前提是回避大回撤

## 七、常见错误自查表（基于回测失效案例）

| 错误 | 典型案例 | 后果 | 修正方法 |
|------|----------|------|----------|
| 假春播 | 2021年抱团瓦解期多次反弹 | 套牢20-30% | 等强势股补跌≥10%+量能验证 |
| 春播一把梭 | 未分仓，单押一只 | 遇倒春寒被动割肉 | 分仓≤30%，2-3个板块 |
| 夏长频繁换股 | 错过主升浪 | 赚小钱亏大钱 | 认定主线就持有，让利润奔跑 |
| 秋收舍不得卖 | 2021年春节前赛道股 | 利润回撤，由盈转亏 | 高潮信号≥2项，逐步兑现 |
| 错过春播追高 | 涨到高位才入场 | 高位站岗 | 等下一轮周期，不追高 |
| 忽视结构性行情 | 2023年AI行情看大盘 | 错过主线 | 增加板块级四季判断 |

## 八、关键参数阈值速查表

| 参数 | 阈值 | 备注 |
|------|------|------|
| **春播-强势股补跌** | ≥10% | 原5%过低，假信号多 |
| **春播-跌停家数** | >50家 | 恐慌情绪确认 |
| **春播-量能验证** | >1.5倍5日均量 | 新增指标，过滤假反弹 |
| **夏长-涨停家数** | >100家 | 赚钱效应确认 |
| **夏长-连板高度** | ≥4板 | 空间打开 |
| **秋收-板块涨幅** | >7% | 单日高潮 |
| **秋收-炸板率** | <20%（过热） | 涨停/炸板>5:1 |
| **冬藏-涨停家数** | <50家 | 情绪冰点 |
| **冬藏-炸板率** | >40% | 赚钱效应差 |
| **仓位-春播** | ≤30% | 试错成本 |
| **仓位-夏长** | ≤70% | 重仓盈利 |
| **仓位-秋收** | ≤20% | 逐步兑现 |
| **仓位-冬藏** | ≤10% | 保命底线 |

**核心结论**：冬藏判断准确率93.78%（空仓保命），春播胜率58.82%（需优化），总体胜率61.9%
**进化方向**：增加量能验证、提高补跌标准、增加板块级判断、明确仓位管理
