### 2.1 核心理念：顺势而为，四季轮回

> **不是预测市场，而是识别市场所处阶段，然后顺势而为**

**从案例图形得出的关键认知：**
1. **四季转换有明确的技术信号**，不是主观臆断
   - 春播→夏长：放量突破底部平台
   - 夏长→秋收：高位滞涨+量能背离
   - 秋收→冬藏：跌破关键支撑位
   - 冬藏→春播：恐慌盘+地量地价

2. **每个阶段持续时间可预期**
   - 春播：3-6周（吸筹需要时间）
   - 夏长：4-8周（主升浪延续）
   - 秋收：2-4周（出货需要时间）
   - 冬藏：4-8周（下跌+筑底）

3. **不逆势操作是保命原则**
   - 冬藏不抄底：不接下跌的飞刀
   - 春播不追高：突破确认再进场
   - 夏长不做T：避免卖飞
   - 秋收不贪心：见好就收

### 2.2 大盘周期判断（定战略仓位）

**春播期（建仓期）：**
```
识别信号（满足3项以上）：
□ 强势股补跌≥10%（前期强势股大幅回调）
□ 跌停家数>30家（情绪恐慌）
□ 涨停家数<40家（情绪冰点）
□ 成交量萎缩至近期低点（地量）
□ 出现恐慌盘（闪崩、多杀多）
□ 个股大面积破净或接近历史低位

关键特征：
- 市场极度悲观，散户恐慌割肉
- 优质股票被错杀，估值回归合理
- 政策底/市场底初现

仓位策略：逐步建仓至30%
```

**夏长期（持有期）：**
```
识别信号（满足3项以上）：
□ 涨停家数>80家（情绪活跃）
□ 出现持续性主线板块（连续3日领涨）
□ 龙头个股连板或趋势加速
□ 中军个股稳健上涨，量能温和放大
□ 跌停家数<15家（情绪稳定）
□ 成交量持续放大

关键特征：
- 板块效应明显，龙头带动跟风
- 中军容纳大资金，趋势稳健
- 赚钱效应扩散，市场情绪乐观

仓位策略：满仓持有，仓位70%
```

**秋收期（兑现期）：**
```
识别信号（满足2项以上）：
□ 后排个股补涨（前期滞涨股突然大涨）
□ 龙头个股高位震荡，涨停不稳定
□ 中军个股放量滞涨（成交量大但涨幅小）
□ 炸板率>30%（封板不坚决）
□ 涨停家数维持高位但连板高度下降
□ 市场情绪亢奋，散户追高热情高涨

关键特征：
- 垃圾股、补涨股活跃，资金恐高
- 龙头股涨不动，需要休整
- 聪明资金开始撤离

仓位策略：逐步减仓至20%
```

**冬藏期（空仓期）：**
```
识别信号（满足2项以上）：
□ 跌停家数>30家（情绪恐慌）
□ 高位股集体杀跌（前期强势股大幅回调）
□ 题材半日游（板块轮动极快，无持续性）
□ 炸板率>40%（打板失败率高）
□ 涨停家数<40家（情绪低迷）
□ 出现多杀多（量化踩踏）

关键特征：
- 市场流动性危机，承接乏力
- 强势股补跌，恐慌盘涌出
- 无明显主线，资金观望

仓位策略：空仓或极低仓位10%
```

### 2.3 板块三要素（定战术主线）

**要素一：龙头（进攻之矛）**

```
识别标准：
1. 涨幅领先：3日涨幅>30%，或5日涨幅>40%
2. 带动性强：同板块跟风股数量>5只
3. 持续性：连续涨停或连阳，回调幅度<5%
4. 地位明确：板块内涨幅第一，市场辨识度高

观察周期：日线+周线共振
- 日线强势突破 + 周线刚启动 = 最佳龙头
- 日线强势但周线高位 = 假龙头（诱多）

案例特征（图形中的夏长阶段）：
- 快速拉升，成交量明显放大
- 带领板块内多只个股上涨
- 回调幅度小，资金承接强
```

**要素二：中军（容纳资金）**

```
识别标准：
1. 市值规模：>200亿（大容量）
2. 流动性：日均成交额>10亿
3. 趋势特征：周线多头排列，稳健上涨
4. 波动性：日波动<7%（适合大资金）
5. 行业地位：细分行业龙头或核心供应商
6. 资金流向：主力资金连续净流入>3日

观察周期：周线为主，日线验证
- 周线低位启动 + 日线量价齐升 = 最佳中军
- 周线高位 + 日线放量滞涨 = 回避

案例特征（图形中的夏长阶段）：
- 涨幅稳健，不涨停但涨不停
- 成交量持续放大，资金持续流入
- 行业地位稳固，业绩支撑
```

**要素三：持续性（空间保障）**

```
判断维度：
1. 政策持续性：政策红利释放周期>3个月
2. 业绩持续性：季报/年报业绩改善趋势明确
3. 资金持续性：主力资金连续净流入>5日
4. 技术持续性：周线级别上升趋势 intact
5. 情绪持续性：涨停梯队完整（1-2-3板都有）
6. 扩散性：产业链上下游联动上涨

观察周期：周线为主
- 周线刚启动（1-3周）+ 资金持续 = 空间大
- 周线上涨>8周 + 资金流出 = 接近尾声

案例特征：
- 不是一日游，而是持续数周的上涨
- 板块内个股轮动上涨，不是单兵突进
- 有政策、业绩、资金多重支撑
```

### 2.4 多周期共振（定执行买卖）

**买点确认（多周期共振）：**

```
最强买点（⭐⭐⭐⭐⭐）：
战略层（月线）：春播期 + 估值低位 ✓
战术层（周线）：板块春播/夏长初期 + 三要素齐备 ✓
执行层（日线）：龙头突破确认 + 量化吸筹 ✓
         ↓
共振强度：⭐⭐⭐⭐⭐
仓位建议：30-50%
持有周期：中线（1-3个月）

次级买点（⭐⭐⭐）：
战略层（月线）：夏长期 ✓
战术层（周线）：板块夏长 + 龙头不倒 ✓
执行层（日线）：回调企稳 + 缩量 ✓
         ↓
共振强度：⭐⭐⭐
仓位建议：10-20%
持有周期：短线（1-2周）
```

**卖点确认（多周期背离）：**

```
清仓卖点（立即执行）：
战略层（月线）：秋收期 + 估值高位 ✓
战术层（周线）：板块秋收 + 三要素瓦解 ✓
执行层（日线）：放量滞涨 + 量化出货 ✓
         ↓
清仓信号：立即减仓至20%以下

减仓信号（部分兑现）：
战术层（周线）：板块涨>6周 ✓
执行层（日线）：60分钟顶背离 ✓
         ↓
减仓信号：兑现50%利润

止损信号（严格执行）：
- 跌破买入价-7%，止损
- 板块跌破20日线，清仓
- 量化席位连续3日净卖出，止损
```

## 四、心法口诀（实战版）

### 春播
> **无恐慌不开仓，无量能不开仓，无龙头不开仓**
> 
> 强势股补跌≥10%，才是真春播
> 分仓布局≤30%，不怕倒春寒
> 春播不看日K看周K，底部震荡要耐心

### 夏长
> **持股待涨≤70%，让利润奔跑**
> 
> 龙头不倒，中军稳健，板块不散
> 日线波动不看，周线趋势不坏不出
> 利用量化做T，日内高抛低吸（可选）
> 高潮信号≥2项，启动秋收

### 秋收
> **去弱留强，逐步兑现至≤20%**
> 
> 后排补涨要警惕，中军滞涨要兑现
> 不追求卖在最高点，吃完鱼身即可
> 量化席位转向日，必须清仓

### 冬藏
> **管住手≤10%，脉冲是诱饵**
> 
> 量化多杀多凶险，严格空仓保命
> 观察量化何时转向，等待真春播
> 冬藏不是一天，是一个阶段

## 六、基于案例图形的实战总结

### 从三张图片学到的核心要点

**1. 春播不是一天，是一个阶段**
- 底部震荡通常持续3-6周
- 会有多次探底，不要想抄到最低点
- 放量突破颈线位才是买点，不是底部

**2. 夏长要拿得住**
- 主升浪沿5日线上行，不破10日线不离场
- 龙头和中军会轮流领涨，不要轻易切换
- 量价齐升时，不要被日内波动甩下车

**3. 秋收要及时兑现**
- 高位放量滞涨就是危险信号
- 双顶形成后果断离场，不要幻想三顶
- 后排补涨是出货信号，不是追涨机会

**4. 冬藏要忍得住**
- 下跌途中不抄底，等待恐慌盘
- 地量地价出现后，再观察春播信号
- 空仓也是一种操作，而且是保命的操作

```
看图形 → 识周期 → 定仓位 → 选板块 → 找买点 → 设止损 → 等卖点

```
