## 一、分析周期分层

```
┌─────────────────────────────────────────────────────────────┐
│  战略层（月线/季度）- 判断大周期位置                          │

│  ├─ 估值水位（全市场PE/PB历史分位）                           │
│  ├─ 资金周期（公募发行/外资流入/融资余额趋势）                  │
│  └─ 政策周期（货币/财政/产业政策方向）                        │
```

## 二、大盘周期分析（战略层）

### 2.1 四季周期定位（月线级别）

**判断标准（多周期验证）：**

| 周期 | 判断指标 | 春播信号 | 夏长信号 | 秋收信号 | 冬藏信号 |
|:---|:---|:---:|:---:|:---:|:---:|
| **估值** | 全A股PE/PB历史分位 | <30% | 30%-70% | >70% | 见顶回落 |
| **资金** | 公募发行/融资余额 | 冰点 | 回升 | 火爆 | 撤离 |
| **政策** | 货币政策基调 | 宽松 | 中性 | 收紧预期 | 紧缩 |
| **情绪** | 月线涨停家数均值 | <50 | 50-100 | >100 | 快速下降 |
| **技术** | 月线MACD | 底背离金叉 | 红柱扩大 | 顶背离 | 死叉绿柱 |

**战略仓位建议：**
- 春播期：逐步建仓，最终仓位30%
- 夏长期：满仓持有，仓位70%
- 秋收期：逐步减仓，最终仓位20%
- 冬藏期：空仓或极低仓位10%

### 2.2 中期趋势判断（周线级别）

**周线三要素：**

1. **趋势方向**
   - 5周均线 > 10周均线 > 20周均线 = 多头排列（夏长）
   - 5周均线上穿20周均线 = 春播转夏长
   - 5周均线下穿10周均线 = 秋收信号
   - 均线空头排列 = 冬藏

2. **量能配合**
   - 周线成交量 > 5周均量 = 健康
   - 周线量价齐升 = 趋势延续
   - 周线量价背离 = 调整预警

3. **结构形态**
   - 周线杯柄形态突破 = 中期买点
   - 周线双底确认 = 春播信号
   - 周线头肩顶 = 秋收信号
   - 周线下降通道 = 冬藏

## 三、板块分析框架（战术层）

### 3.1 板块三要素验证

**要素一：龙头股（进攻先锋）**
```
识别标准：
├─ 涨幅：3日涨幅>30% 或 周线涨幅>20%
├─ 强度：连续涨停或连阳，回调幅度<5%
├─ 带动性：同板块跟风股数量>5只
├─ 持续性：龙头不倒，板块不散
└─ 量化验证：龙虎榜显示机构+量化共同介入

观察周期：日线+周线共振
- 日线突破前高 + 周线突破平台 = 强龙头
- 日线强势但周线滞涨 = 假龙头
```

**要素二：中军股（容纳大资金）**
```
识别标准：
├─ 市值：>200亿（大容量）
├─ 流动性：日均成交额>10亿
├─ 趋势：周线多头排列，稳健上涨
├─ 波动：日波动<7%（适合大资金）
├─ 地位：行业细分龙头或核心供应商
└─ 资金：主力资金连续净流入>3日

观察周期：日线趋势 + 周线位置
- 周线低位启动 + 日线量价齐升 = 最佳中军
- 周线高位 + 日线放量滞涨 = 回避
```

**要素三：持续性（上升空间）**
```
判断维度：
├─ 政策持续性：政策红利释放周期>3个月
├─ 业绩持续性：季报/年报业绩改善趋势
├─ 资金持续性：主力资金连续净流入>5日
├─ 技术持续性：周线级别上升趋势 intact
├─ 情绪持续性：涨停梯队完整（1-2-3板）
└─ 扩散性：产业链上下游联动上涨

观察周期：周线为主，日线验证
- 周线刚启动（1-3周）+ 资金持续 = 空间大
- 周线上涨>8周 + 资金流出 = 接近尾声
```

### 3.2 板块四季定位

**板块春播（建仓期）：**
- 周线底部震荡>4周，成交量萎缩
- 龙头开始异动，首板涨停试探
- 中军小幅建仓，阳线多于阴线
- 政策预期酝酿，未全面发酵

**板块夏长（主升浪）：**
- 周线突破平台，成交量放大
- 龙头连板或趋势加速
- 中军稳健上涨，量价齐升
- 板块涨停家数维持高位（>10家）

**板块秋收（兑现期）：**
- 周线连续上涨>6周，乖离率过大
- 龙头高位震荡，涨停不稳定
- 中军放量滞涨，长上影线增多
- 后排补涨股活跃（资金恐高）

**板块冬藏（调整期）：**
- 周线破位，跌破10周线
- 龙头跌停或大幅回调
- 中军连续阴跌，资金撤离
- 板块涨停家数<3家

## 四、多周期共振交易系统

### 4.1 买点确认（多周期共振）

**共振买点（最强信号）：**
```
战略层（月线）：春播期 + 估值低位 ✓
战术层（周线）：板块春播/夏长初期 + 三要素齐备 ✓
执行层（日线）：突破确认 + 量化吸筹 ✓
         ↓
共振强度：⭐⭐⭐⭐⭐
仓位建议：30-50%
```

**次级买点（一般信号）：**
```
战略层（月线）：夏长期 ✓
战术层（周线）：板块夏长 + 龙头不倒 ✓
执行层（日线）：回调企稳 + 缩量 ✓
         ↓
共振强度：⭐⭐⭐
仓位建议：10-20%
```

### 4.2 卖点确认（多周期背离）

**共振卖点（清仓信号）：**
```
战略层（月线）：秋收期 + 估值高位 ✓
战术层（周线）：板块秋收 + 三要素瓦解 ✓
执行层（日线）：放量滞涨 + 量化出货 ✓
         ↓
清仓信号：立即减仓至20%以下
```

**减仓信号（部分兑现）：**
```
战术层（周线）：板块涨>6周 ✓
执行层（日线）：60分钟顶背离 ✓
         ↓
减仓信号：兑现50%利润
```

## 七、心法升级（多周期版）

> **战略定方向，战术定板块，执行定买卖点**

**春播期（战略建仓）：**
- 月线低位 + 周线底部 = 大胆布局
- 寻找刚启动的板块（周线1-3周）
- 仓位逐步加至30%，不追高大涨股

**夏长期（战术持有）：**
- 板块三要素 intact = 坚定持有
- 龙头不倒，中军稳健 = 不动如山
- 日线波动不看，周线趋势不坏不出

**秋收期（战略减仓）：**
- 板块周线>6周 + 后排补涨 = 兑现
- 龙头开始滞涨 + 中军放量 = 减仓
- 逐步降至20%，不贪心最后一段

**冬藏期（战略空仓）：**
- 板块破位 + 三要素瓦解 = 彻底离场
- 不抄底，不抢反弹，不逆周期
- 等待下一个春播信号（月线+周线双底）
