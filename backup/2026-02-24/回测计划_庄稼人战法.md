# 庄稼人战法 · 五年回测计划（2021-2025）

## 回测目标
验证战法在以下市场环境中的有效性，找出进化方向：
- 2021年：机构赛道抱团瓦解、量化开始兴起
- 2022年：疫情反复、地缘冲突、市场大幅波动
- 2023年：AI元年、ChatGPT引爆、市场结构性分化
- 2024年：新国九条、监管趋严、微盘股危机
- 2025年：当前市场环境

## 回测维度

### 1. 四季轮回识别准确率
- 统计每个"季节"判断的正确率
- 找出假春播、假秋收的识别特征
- 量化：胜率、盈亏比、最大回撤

### 2. 主线板块识别
- 每轮周期的真实主线 vs 战法筛选出的主线
- 三要素（天时地利人和）的权重优化
- 失效案例：为什么有些符合三要素的板块没走出来？

### 3. 市场进化特征
- 量化私募兴起后的影响（2021年后）
- 20cm/30cm制度变化的影响
- 信息传播速度加快的影响（抖音/雪球）
- 机构vs游资博弈模式变化

### 4. 方法论优化方向
- 春播：恐慌度量的量化标准（跌停家数阈值？）
- 夏长：补涨挖掘的时效性窗口
- 秋收：高潮识别的多维度指标
- 冬藏：强势股补跌的确认信号

## 数据来源
- 东方财富API（已验证可用）
- 涨停/跌停/炸板数据
- 板块指数历史数据
- 个股连板数据

## 执行计划

### Phase 1：数据收集（1-2天）
- 收集2021-2025每日涨停池、跌停池数据
- 收集主要板块指数历史走势
- 标记历史上已知的主线板块（人工标注）

### Phase 2：回测执行（2-3天）
- 按战法规则逐日推演
- 记录每个判断的盈亏结果
- 统计胜率、盈亏比、最大回撤

### Phase 3：分析与优化（1-2天）
- 找出失效模式和原因
- 提出方法论改进建议
- 更新实战分析模板

## 预期产出
1. 五年回测报告（含胜率统计）
2. 战法进化版方法论
3. 更新的每日分析模板
4. 关键参数阈值表（春播/秋收的量化标准）
