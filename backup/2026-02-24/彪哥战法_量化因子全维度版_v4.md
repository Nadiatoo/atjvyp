## 一、动量识别法（量化趋势跟踪）

### 1.1 量化常用的动量策略
- **短期动量**（1-5日）：追涨杀跌
- **中期动量**（10-20日）：趋势跟踪
- **长期动量**（60日+）：风格轮动

### 1.2 我们识别量化动量的方法

**动量加速度异常**
```
特征：
- 价格变化速度突然加快（>2倍近期平均）
- 但无基本面/消息驱动
- 多股同步加速（量化批量操作）

判断：
✅ 建仓信号：低位动量加速 + 量能配合 + 板块共振
❌ 出货信号：高位动量加速 + 量能背离 + 无量化席位跟进

案例识别：
- 量化动量建仓：3日内涨幅15%+，分时阶梯式，换手温和放大
- 量化动量出货：1日内涨幅>8%后回落，长上影线，换手突增
```

**动量持续性检测**
```
量化算法特征：
- 动量衰减后自动减仓（程序化）
- 动量持续则加仓（趋势跟踪）

我们利用：
- 观察动量衰减信号：连板断板后次日低开概率
- 量化减仓导致的低点是低吸机会（非真正走弱）
- 动量恢复时跟进（量化重新加仓）

阈值：
- 动量衰减：3日内涨幅从>15%降至<5%
- 动量恢复：缩量整理后再度放量突破
```

**多时间框架动量背离**
```
量化陷阱：
- 日线动量强劲（吸引跟风）
- 但60分钟动量已衰竭（量化准备出货）

识别方法：
- 日线MACD红柱，但60分钟MACD绿柱
- 日线成交量放大，但60分钟成交量萎缩
- 日线创新高，但60分钟未创新高

应对：
- 多时间框架背离 = 量化出货前兆
- 不追高，等待回踩确认
```

## 二、量能识别法（成交量异常检测）

### 2.1 量化常用的量能策略
- **放量突破**：突破关键位时放量跟进
- **缩量整理**：洗盘/锁仓检测
- **量能背离**：量价背离预警

### 2.2 扩展识别维度

**量能分布形态**
```
量化建仓量能特征：
- 阶梯式放量（非一次性爆发）
- 放量日涨幅温和（3-5%），非涨停
- 缩量日跌幅极小（洗盘温和）

量化出货量能特征：
- 脉冲式放量（间歇性突增）
- 放量日滞涨或下跌（对倒出货）
- 缩量日阴跌（无量化托盘）

识别公式：
量能健康度 = (放量日涨幅 / 放量倍数) / (缩量日跌幅 / 缩量倍数)
> 1.5 健康（建仓期）
< 0.5 危险（出货期）
```

**换手率分层分析**
```
量化关注的换手区间：
┌─────────────┬─────────────┬────────────────┐
│  换手率区间  │  量化策略   │    识别信号     │
├─────────────┼─────────────┼────────────────┤
│   < 3%     │  锁仓观察   │  量化吸筹完毕   │
│  3% - 8%   │  积极做T   │  量化活跃期    │
│  8% - 15%  │  高抛低吸  │  量化控盘期    │
│  15% - 25% │  出货/建仓 │  关键决策点    │
│   > 25%    │  边拉边出  │  出货嫌疑大    │
└─────────────┴─────────────┴────────────────┘

应用：
- 从<3%突增至15%+：量化建仓启动
- 维持在8-15%：量化做T阶段，可持有
- 从15%+突增至25%+：量化出货，减仓
```

**量比与委比组合**
```
量比 = 当前成交量 / 过去5日平均成交量
委比 = (委买手数 - 委卖手数) / (委买手数 + 委卖手数)

量化行为识别矩阵：
┌──────────┬──────────┬─────────────────────┐
│   量比   │   委比   │     量化行为解读     │
├──────────┼──────────┼─────────────────────┤
│   > 2    │   > 30%  │  量化主动吸筹        │
│   > 2    │  < -30%  │  量化主动出货        │
│  1-2     │   > 20%  │  量化托底做T         │
│  1-2     │  < -20%  │  量化压盘洗盘        │
│   < 1    │  接近0   │  量化锁仓观望        │
└──────────┴──────────┴─────────────────────┘
```

## 三、波动识别法（波动率交易识别）

### 3.1 量化常用的波动率策略
- **波动率突破**：高波动时追涨杀跌
- **波动率收敛**：低波动时布局突破
- **波动率套利**：跨品种波动率差异

### 3.2 波动率识别维度

**ATR（真实波幅）异常**
```
ATR = Max(High-Low, |High-Close昨日|, |Low-Close昨日|)

量化建仓期特征：
- ATR从低位（<3%）逐步扩大至5-8%
- 但不超过10%（防止过度波动）
- ATR扩大时成交量同步放大

量化出货期特征：
- ATR突然扩大至>10%
- 长上下影线增多
- ATR扩大但成交量萎缩（无量化承接）

应用：
- ATR 3-5%：量化吸筹期，可介入
- ATR 5-8%：量化活跃期，持有
- ATR >10%：量化出货期，减仓
- ATR <2%：量化锁仓期，观望
```

**日内波动率分布**
```
量化日内交易特征：
- 开盘后30分钟波动最大（量化调仓）
- 10:00-11:30波动中等（量化做T）
- 13:00-14:00波动较小（量化观望）
- 14:30后波动再次放大（量化尾盘操作）

识别量化日内模式：
- 早盘急拉后横盘：量化出货型
- 早盘急跌后横盘：量化吸筹型
- 全天锯齿形震荡：量化做T型
- 尾盘急拉：量化诱多型

应对：
- 早盘不急追，观察30分钟
- 尾盘不追高，防量化诱多
```

## 四、盘口识别法（订单簿分析扩展）

### 4.1 原有维度扩展

**盘口深度不平衡**
```
量化行为：
- 买盘深度 >> 卖盘深度：量化托底（但可能是诱多）
- 卖盘深度 >> 买盘深度：量化压盘（但可能是诱空）
- 买卖盘深度接近：量化平衡市

识别真假：
真托底：买盘挂单大且稳定，成交后迅速补单
假托底：买盘挂单大但易撤，成交后不补单

量化压盘吸筹：
- 卖盘挂大单压价
- 但买盘有承接不跌
- 实际在悄悄吸筹
```

**冰山订单识别**
```
量化隐藏大单手法：
- 显示100手，实际有10000手在队列后
- 通过算法分批暴露

识别迹象：
- 某一价位反复成交但挂单不见减少
- 成交后同一价位迅速补单
- 大单成交时价格冲击小（被冰山承接）

应用：
- 发现冰山买盘 = 量化建仓迹象
- 发现冰山卖盘 = 量化出货迹象
```

## 五、周期识别法（算法周期探测）

### 5.1 量化常用的周期策略
- **算法调仓周期**：日度/周度/月度
- **策略轮换周期**：动量/反转/均值回归切换
- **参数优化周期**：定期重新训练模型

### 5.2 周期识别维度

**量化调仓周期**
```
常见量化调仓时点：
- 每日开盘前（9:15-9:25）：集合竞价调仓
- 每日收盘前（14:50-15:00）：尾盘调仓
- 每周一开盘：周度调仓
- 每月月初：月度调仓

识别方法：
- 观察这些时段的异常成交量
- 多股同步异动（批量调仓）
- 价格冲击后快速修复（算法滑点控制）

利用：
- 调仓前埋伏 = 跟随量化
- 调仓时反向操作 = 利用量化冲击
- 调仓后跟随 = 确认量化方向
```

**算法周期共振**
```
量化策略周期性：
- 动量策略周期：5-20日
- 均值回归策略周期：1-5日
- 趋势跟踪策略周期：20-60日

识别当前主导周期：
- 5日内涨幅>20%股数量多 = 动量周期
- 大涨次日大跌股多 = 均值回归周期
- 持续上涨1月+ = 趋势周期

应用：
- 动量周期：追涨策略有效
- 均值回归周期：高抛低吸
- 趋势周期：持有不动
```

## 六、相对识别法（板块相对强弱）

### 6.1 量化常用的横截面策略
- **行业轮动**：强行业vs弱行业
- **风格轮动**：大盘vs小盘，价值vs成长
- **个股选择**：行业内相对强弱

### 6.2 相对识别维度

**行业内相对强弱（RS）**
```
RS = 个股涨幅 / 行业平均涨幅

量化选股逻辑：
- RS > 1.2：行业内强势股（量化偏好）
- RS 0.8-1.2：行业平均水平
- RS < 0.8：行业内弱势股（量化回避）

识别量化建仓：
- 个股RS从<0.8突增至>1.2
- 行业下跌时个股抗跌（RS提升）
- 量化正在建仓该个股

应用：
- 选板块内RS快速提升股
- 回避RS持续下降股
```

**风格暴露识别**
```
量化风格因子：
- 市值因子：大盘/中盘/小盘
- 估值因子：价值/成长
- 动量因子：高动量/低动量
- 波动因子：高波动/低波动

识别当前主导风格：
- 涨停股平均市值 > 500亿 = 大盘风格
- 涨停股平均市值 < 100亿 = 小盘风格
- 低PE股涨停多 = 价值风格
- 高PE股涨停多 = 成长风格

应用：
- 跟随量化风格切换
- 不逆风格操作
```

## 七、资金识别法（主力流向追踪）

### 7.1 量化常用的资金流策略
- **大单追踪**：跟随主力资金
- **北向资金**：外资动向
- **融资余额**：杠杆资金情绪

### 7.2 资金识别维度

**主力资金分层**
```
资金层级识别：
┌─────────────┬─────────────┬─────────────────┐
│   资金类型   │   单笔规模   │     量化特征     │
├─────────────┼─────────────┼─────────────────┤
│   散户      │   < 50万    │  分散，情绪化    │
│   中户      │  50-200万   │  有一定规律     │
│   大户      │  200-1000万 │  可能是量化分拆  │
│   机构     │  > 1000万   │  机构或量化大单  │
│   量化      │  拆单隐藏   │  需通过席位识别  │
└─────────────┴─────────────┴─────────────────┘

识别量化资金：
- 中单（50-200万）持续流入 = 量化建仓
- 大单（>1000万）突然流出 = 机构出货
- 散户资金流入但股价跌 = 量化出货
```

**融资余额变化率**
```
融资余额增速 = (当日融资余额 - 昨日融资余额) / 昨日融资余额

量化利用融资特征：
- 融资余额快速增加 = 散户情绪高涨（量化出货时机）
- 融资余额快速减少 = 散户恐慌（量化吸筹时机）
- 融资余额稳定 = 散户观望（量化布局期）

阈值：
- 日增速 > 5% = 散户情绪过热，量化可能出货
- 日增速 < -5% = 散户情绪冰点，量化可能吸筹
```

## 八、情绪识别法（散户/机构情绪）

### 8.1 量化常用的情绪策略
- **情绪反转**：极端情绪反向操作
- **情绪动量**：情绪延续跟随
- **情绪分歧**：分歧时布局

### 8.2 情绪识别维度

**涨跌停家数情绪**
```
已有指标扩展：
- 涨停家数 / 总交易家数 = 涨停率
- 跌停家数 / 总交易家数 = 跌停率
- 涨停率 / 跌停率 = 情绪比值

量化情绪周期：
┌─────────┬──────────┬──────────┬────────────────┐
│  阶段   │  涨停率   │  跌停率   │    量化操作     │
├─────────┼──────────┼──────────┼────────────────┤
│  冰点   │   < 1%   │   > 2%   │    建仓吸筹    │
│  复苏   │  1%-3%   │   < 1%   │    加速建仓    │
│  高潮   │   > 5%   │   < 0.5% │    开始出货    │
│  退潮   │  3%-5%   │  0.5-1%  │    加速出货    │
│  杀跌   │   < 2%   │   > 1%   │    观望等待    │
└─────────┴──────────┴──────────┴────────────────┘
```

**炸板率与回封率**
```
炸板率 = 炸板家数 / (涨停家数 + 炸板家数)
回封率 = 回封家数 / 炸板家数

量化行为：
- 炸板率高（>40%）+ 回封率低（<30%）= 量化出货
- 炸板率高（>40%）+ 回封率高（>50%）= 量化洗盘
- 炸板率低（<20%）= 量化锁仓，情绪一致

应用：
- 炸板率高+回封率高 = 分歧中走强，可参与
- 炸板率高+回封率低 = 量化出货，回避
```

## 九、模式识别法（算法模式探测）

### 9.1 量化常用的机器学习策略
- **价格模式识别**：历史形态匹配
- **异常检测**：离群点识别
- **聚类分析**：股票分组交易

### 9.2 模式识别维度

**历史形态匹配**
```
量化算法会匹配历史相似K线形态

常见匹配模式：
- V型底反转
- 杯柄形态
- 三角形收敛
- 旗形整理

识别量化匹配：
- 股价运行至关键形态位时突然加速
- 多股同步出现相似形态突破
- 突破时成交量配合完美（量化设计）

应用：
- 识别量化关注的形态位
- 在形态突破前埋伏
- 不追高突破（可能是量化诱多）
```

**聚类效应识别**
```
量化聚类交易特征：
- 同一聚类股票同步涨跌
- 聚类内股票轮动（补涨）

常见聚类维度：
- 行业聚类
- 市值聚类
- 动量聚类
- 波动率聚类

识别量化聚类：
- 无明显关联的股票同步异动
- 某属性股票集体走强
- 量化席位在聚类内多股出现

应用：
- 发现量化关注的聚类
- 在聚类内寻找补涨机会
- 回避量化正在出货的聚类
```

## 十、事件识别法（信息优势检测）

### 10.1 量化常用的事件驱动策略
- **业绩超预期**：财报公告交易
- **宏观数据**：经济数据发布交易
- **政策变化**：政策解读交易

### 10.2 事件识别维度

**信息优势识别**
```
量化信息优势来源：
- 高频数据（卫星、消费等另类数据）
- 自然语言处理（新闻、公告快速解读）
- 供应链数据（上下游提前知晓）

识别量化信息优势：
- 无公开消息时股价异动
- 消息公布前成交量异常
- 消息公布后量化快速反应（毫秒级）

应对：
- 不追逐信息优势
- 识别量化信息交易后的持续性
- 信息公布后观察量化是否延续
```

**日历效应识别**
```
量化常用的日历效应：
- 月初效应：月初上涨概率高
- 月末效应：月末调整概率高
- 假期效应：假期前减仓，假期后加仓
- 财报季效应：财报前布局，财报后兑现

识别量化日历交易：
- 特定时间点的规律性行为
- 多股同步的日历模式
- 量化席位在特定时期活跃

应用：
- 提前布局日历效应
- 利用量化日历交易规律
```

## 十一、回归识别法（反转策略识别）

### 11.1 量化常用的均值回归策略
- **统计套利**：配对交易
- **反转交易**：超跌反弹
- **布林带回归**：触及轨道回归

### 11.2 回归识别维度

**统计套利识别**
```
量化配对交易特征：
- 两只相关股票价差异常扩大
- 量化同时买入弱势股，卖出强势股
- 价差回归时平仓

识别量化配对：
- 行业内两只龙头股走势背离
- 背离后同步出现量化席位
- 随后价差收窄

应用：
- 发现量化配对交易
- 跟随量化做多弱势股
- 不逆势追涨强势股
```

**布林带回归**
```
布林带构成：
- 中轨 = 20日均线
- 上轨 = 中轨 + 2倍标准差
- 下轨 = 中轨 - 2倍标准差

量化布林带策略：
- 触及上轨 = 卖出（回归中轨）
- 触及下轨 = 买入（回归中轨）
- 突破上轨 = 追涨（趋势延续）

识别量化布林带操作：
- 股价触及上轨后回落 = 量化卖出
- 股价触及下轨后反弹 = 量化买入
- 突破上轨持续涨 = 量化追涨

应用：
- 下轨附近 = 量化买入区（可跟随）
- 上轨附近 = 量化卖出区（可减仓）
```

## 十二、联动识别法（跨市场套利追踪）

### 12.1 量化常用的跨市场策略
- **AH股套利**：A股与H股价差
- **期现套利**：期货与现货
- **跨品种套利**：相关品种价差

### 12.2 联动识别维度

**AH股联动**
```
量化AH套利特征：
- AH溢价率异常时操作
- A股涨H股跌 = 量化卖出A股
- A股跌H股涨 = 量化买入A股

识别量化AH操作：
- AH股走势背离
- 背离时量化席位活跃
- 随后溢价率回归

应用：
- AH溢价>130% = A股高估，量化可能卖出
- AH溢价<110% = A股低估，量化可能买入
```

**期现联动**
```
股指期货与现货关系：
- 期货升水 = 看涨情绪
- 期货贴水 = 看跌情绪
- 期现背离 = 套利机会

量化期现套利：
- 期货升水过大 = 卖期货买现货
- 期货贴水过大 = 买期货卖现货

识别量化期现操作：
- 期现价差异常时现货异动
- 尾盘期现同步操作（量化调仓）

应用：
- 期货大幅升水 = 量化可能在卖出现货
- 期货大幅贴水 = 量化可能在买入现货
```

## 十四、心法升级（12维度版）

### 春播
- 无恐慌不开仓，无量能不开仓，无量化吸筹不开仓
- 强势股补跌≥10% + 量化锯齿形吸筹 = 真春播
- **12维度共振≥4个 = 确认春播信号**
- 分仓布局≤30%，不怕倒春寒

### 夏长
- 持股待涨≤70%，让利润奔跑
- **动量持续 + 量化做T + 情绪稳定 = 夏长延续**
- 利用12维度评分，80分以上持有，60分以下减仓
- 高潮信号≥2项 + 情绪指数>0.8 = 启动秋收

### 秋收
- 去弱留强，逐步兑现至≤20%
- **多维度背离≥3个 = 量化出货确认**
- 不追求卖在最高点，12维度评分<40分清仓
- 对倒盘 + 情绪指数>0.9 + 期现背离 = 必须离场

### 冬藏
- 管住手≤10%，脉冲是诱饵
- **12维度评分<20分 = 绝对空仓**
- 量化多杀多凶险，情绪冰点≠立即抄底
- 等待12维度共振≥4个，才是春播时机
