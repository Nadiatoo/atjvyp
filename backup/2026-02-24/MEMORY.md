### 方法论进化v4.3（近五年复盘与量化进化）：已完成
- **复盘周期**：2021-2025年市场环境演变
- **量化占比变化**：15%（2021）→30%（2025）
- **三大失效案例分析**：
  1. **2023年AI行情**：传统秋收信号后，AI继续上涨3个月
     - 原因：量化资金持续流入，板块独立周期
     - 进化：板块独立四季判断，量化席位持续流入则延长持有
  2. **2024年1月微盘踩踏**：春播信号后，继续暴跌30%
     - 原因：量化踩踏规模超预期，春播期延长（4周→12周）
     - 进化：强势股补跌标准≥10%→≥15%，分批建仓机制（首仓10%）
  3. **2022年新能源反复**：秋收信号后，调整2周继续上涨
     - 原因：量化做T策略，机构抱团
     - 进化：夏长期做T策略（核心仓位50%+做T仓位20%），震荡市允许20%回调
- **核心进化**：从"散户情绪周期"到"量化资金周期"
  - 散户情绪指标权重：85%→60%
  - 量化资金指标权重：15%→40%
- **量化识别四维体系**：
  1. 盘口识别：大单夹小单、对倒盘、锯齿形
  2. 分时识别：织布机、冲击波
  3. 量能识别：间歇性放量、异常缩量
  4. 龙虎榜识别：量化席位行为、机构+量化混合信号
- **四季标准量化适配**：
  - 春播：补跌≥15%（原≥10%），跌停>50家（原>30家）
  - 夏长：涨停>80家（原>100家），连板≥3板（原≥4板）
  - 秋收：炸板率>35%（原>30%），量化净卖出>2日
  - 冬藏：跌停>50家（原>30家），量化多杀多识别
- **核心心法**：市场永远是对的，战法必须持续进化

### 核心心法（庄稼人战法精髓）

**1. 顺势而为，四季轮回**
> 不是预测市场，而是识别市场所处阶段，然后顺势而为

**2. 不逆势操作原则**
- 冬藏不抄底：不接下跌的飞刀，空仓保命
- 春播不追高：突破确认再进场，不怕错过
- 夏长不做T：避免卖飞，让利润奔跑
- 秋收不贪心：吃完鱼身即可，不求最高点

**3. 看图形识周期（基于实战案例）**
```
价格
  │                           【秋收兑现点】
  │                           双顶形成+跌破颈线
  │                           ↓
  │                    ┌───┐ ← 第二顶（量能背离）
  │                   /     \
  │                  /       \ ← 第一顶
  │                 /         \
  │            ┌───┐           \
  │           /     \           └───┐
  │          /       \               \
  │         /         \               \
  │    ┌───┐           \               \
  │   /     \  【春播建仓点】      【夏长持股】
  │  /       \  放量突破颈线 → 沿5日线上行
  │ /         \      ↑
  │/           \     │
  └─────────────────────────────────────────→ 时间
       【冬藏空仓】   【春播震荡期】
       等待恐慌盘    锯齿形吸筹
```

**4. 多周期共振（定买卖）**
- 战略层（月线）：定四季大方向
- 战术层（周线）：定板块主线
- 执行层（日线）：定具体买卖点

**5. 板块三要素验证**
- 龙头：3日涨幅>30%，带动性>5只跟风股
- 中军：市值>200亿，日均成交>10亿，周线稳健
- 持续性：政策>3月 + 资金连续流入>5日 + 产业链联动

**6. 量化时代适应（v4.3核心）**
- 从"散户情绪周期"到"量化资金周期"
- 散户情绪：涨停/炸板/跌停（权重60%）
- 量化资金：龙虎榜席位/大单流向/算法特征（权重40%）
- 利用量化波动，而非被其收割
