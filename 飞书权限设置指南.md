### 步骤2：进入飞书开放平台

1. 打开浏览器访问：https://open.feishu.cn/app
2. 扫码或密码登录你的飞书账号
3. 在应用列表中找到你的应用
   - 应用ID: `cli_a91100b306785cb2`
   - 或你自定义的应用名称

### 步骤3：进入权限管理页面

1. 点击应用名称进入详情
2. 左侧菜单找到 **「权限管理」**
3. 点击右上角 **「批量导入」** 按钮

### 步骤4：粘贴JSON代码

1. 在弹出的对话框中
2. 删除原有内容
3. 粘贴上面的完整JSON代码
4. 点击 **「确定」**

### 步骤5：发布新版本

1. 左侧菜单点击 **「版本管理与发布」**
2. 点击 **「创建版本」** 按钮
3. 填写版本信息：
   - 版本号: `1.0.1`
   - 更新说明: `添加云文档和云盘写入权限，支持自动保存报告`
4. 点击 **「保存并发布」**
5. 等待审核（企业内部应用通常自动通过）

### 步骤6：重新授权（重要！）

**方式A：删除重新添加**
1. 打开飞书手机APP或电脑端
2. 找到机器人聊天窗口
3. 点击右上角「...」→「删除会话」
4. 重新搜索机器人并添加

**方式B：管理员后台刷新**
1. 让飞书管理员进入管理后台
2. 找到「应用管理」
3. 找到你的应用，点击「刷新权限」

如果批量导入失败，可以手动添加：

```
飞书开放平台 → 你的应用 → 权限管理 → 批量开通
```

**云盘权限（drive）：**

- [x] `drive:folder:read` - 文件夹读取

**知识库权限（wiki）：**

## ⚠️ 常见问题

### Q1: 批量导入提示格式错误？
**解决：** 确保JSON格式正确，没有多余的逗号，使用英文引号。

### Q2: 权限已添加但无法使用？
**解决：** 必须「发布新版本」并「重新授权」才能生效。

### Q3: 找不到「批量导入」按钮？
**解决：** 某些旧版界面可能没有，请使用「方法二：手动逐个添加」。

### Q4: 发布版本后一直显示「审核中」？
**解决：** 
- 企业内部自建应用通常自动通过（几分钟内）
- 如果是第三方应用，需要管理员审核
- 可以联系飞书管理员加速审核
