## 一、战法核心理念

### 1.1 市场周期律

股市如四季，有春耕、夏长、秋收、冬藏。顺应周期，胜率自然高。

```
四季轮回图：

    冬藏 ❄️          春播 🌱
    空仓保命          恐惧中播种
         ↖         ↗
           周期律
         ↙         ↘
    秋收 🍂          夏长 ☀️
    贪婪中兑现        趋势中持仓
```

### 3.1 春播期：数据验证的恐慌底部

#### 春播确认信号（v4.0标准）

**必须满足条件（≥4项）：**

| 序号 | 信号 | 数据来源 | 阈值 | 权重 |
|-----|------|---------|------|------|
| 1 | 强势股补跌 | Tushare日线 | ≥10% | ★★★★☆ |
| 2 | 跌停家数激增 | Tushare limit_list | >50家 | ★★★★★ |
| 3 | 成交量放大 | Tushare daily | >1.5倍5日均量 | ★★★★☆ |
| 4 | 恐慌指数 | 计算指标 | >70 | ★★★☆☆ |
| 5 | 量化席位净卖出峰值 | Tushare top_list | 连续3日净卖出 | ★★★★★ |
| 6 | 机构首次净买入 | Tushare top_inst | 单日前10大买入 | ★★★★☆ |
| 7 | 板块共振 | Tushare概念板块 | ≥3个板块同步反弹 | ★★★☆☆ |

**春播信号等级**
```
🟢 强春播（5项以上）：重仓介入
🟡 中春播（4项）：标准建仓
🟠 弱春播（3项）：小仓位试探
🔴 无信号（<3项）：观望
```

#### 春播建仓策略

**仓位管理**
```
标准春播：30%仓位
强春播：   35%仓位
弱春播：   15%仓位

建仓节奏（以30%为例）：
Day1: 信号确认，建仓10%
Day2: 数据验证（跌停减少），加至20%
Day3: 反弹确认，加满至30%
```

**选股标准（春播三要素）**
```
1. 错杀绩优股
   - 基本面无变化
   - 因系统性风险下跌
   - Tushare财务指标ROE>10%

2. 主线先跌先弹
   - 板块跌幅排名前列
   - Tushare概念板块跌幅>15%
   - 龙虎榜机构开始买入

3. 机构+量化净流入
   - Tushare top_inst机构净买入
   - top_list量化席位转向
   - 北向资金连续流入
```

**止损纪律**
```
硬性止损：
- 跌破春播低点-7%，无条件止损
- 量化席位连续3日净卖出，止损
- 跌停家数再次>50家，清仓

时间止损：
- 春播后10日无反弹，减仓50%
- 春播后20日无反弹，清仓
```

#### 夏长持仓策略

**仓位管理**
```
标准夏长：70%仓位
强夏长：   70%仓位（满仓）
弱夏长：   40%仓位

持仓策略：
- 核心仓位（50%）：趋势中军，不做T
- 灵活仓位（20%）：高抛低吸
- 现金储备（30%）：等待更好机会
```

**量化做T适应策略**
```
识别量化做T（锯齿形分时）：
- 日内波动>8%（正常现象）
- 不因此恐慌卖出
- 可利用波动高抛低吸

避免的错误：
- ❌ 看到放量就卖出（可能是量化对倒）
- ❌ 日内回调就止损（正常波动）
- ✅ 只看收盘趋势，忽略日内噪音
```

**板块阵型监控**
```
先锋（连板龙头）：
- Tushare limit_list连板高度
- 量化席位净买入为主
- 断板后走趋势而非A杀

中军（大市值核心）：
- Tushare top_inst机构净买入
- 量化席位做T痕迹
- 均线多头排列

后排（低位补涨）：
- Tushare概念板块低位股
- 量化席位新进
- 快速补涨后快速出货
```

**撤退信号（秋收预警）**
```
必须撤退（出现即减仓）：
- 量化席位连续2日净卖出
- 连板高度无法突破5板
- 炸板率>50%

考虑撤退（结合其他信号）：
- 板块内后排股开始滞涨
- 跟风分支鸡犬升天
- 千亿市值涨停
```

#### 秋收兑现策略

**分档兑现**
```
第一档（高潮日）：
- 清仓所有跟风补涨股
- 减仓量化席位净卖出的个股
- 保留机构主导的核心股

第二档（量化出货确认）：
- 对倒盘出现，再减50%
- 量化连续2日净卖出，清仓该个股
- 总体仓位降至≤20%

第三档（趋势破位）：
- 跌破10日线，清仓
- 量化席位全面撤退，彻底离场
- 不等待反包（量化时代反包减少）
```

**量化出货识别要点**
```
对倒出货特征：
- Tushare top_list买卖金额接近
- 成交量放大但价格不涨
- 分时呈现"锯齿形"出货

真突破 vs 假突破：
- 真突破：量化净买入+机构买入+板块共振
- 假突破：量化对倒+无量+无板块
```

### 4.2 量化行为识别矩阵

| 量化行为 | 识别特征 | 数据来源 | 对战法影响 |
|---------|---------|---------|-----------|
| **量化吸筹** | 低位锯齿形+间歇放量+净买入 | Tushare日线+top_list | 春播建仓信号 |
| **量化拉升** | 阶梯上涨+净买入+机构混合 | Tushare日线+top_inst | 夏长持仓信号 |
| **量化做T** | 日内锯齿形+买卖均衡 | Tushare top_list | 正常现象，忽略 |
| **量化对倒** | 买卖金额接近+价格横盘 | Tushare top_list | 秋收撤退信号 |
| **量化出货** | 高位放量+大额净卖出 | Tushare top_list | 秋收清仓信号 |
| **量化多杀多** | 集体净卖出+闪崩 | Tushare top_list | 冬藏空仓信号 |

### 4.3 机构+量化混合信号

**最佳组合信号**
```
🟢 机构建仓 + 量化做T = 最佳持仓期
   - 机构买入：基本面认可
   - 量化做T：提供流动性
   - 策略：跟随机构，耐心持有

🟡 机构观望 + 量化买入 = 观察期
   - 量化短炒可能性大
   - 策略：小仓位参与，快进快出

🟠 机构卖出 + 量化买入 = 危险期
   - 量化对倒出货嫌疑
   - 策略：不参与，观望

🔴 机构卖出 + 量化卖出 = 绝对回避
   - 无承接盘
   - 策略：空仓，不抄底
```

## 七、心法口诀（v4.0数据版）

### 春播
- 无恐慌不开仓，无量化净卖不出手
- 强势股补跌≥10%，Tushare跌停数>50
- 量化席位卖出峰值，机构开始净买入
- 数据验证再进场，分仓布局≤35%

### 夏长
- 持股待涨≤70%，量化净买是友军
- 阶梯上涨是常态，连板高度五板顶
- 机构量化混合最佳，锯齿波动莫慌张
- Tushare数据监控，高潮信号启动秋收

### 秋收
- 去弱留强，量化净卖即减仓
- 对倒出货要识别，Tushare买卖金额比
- 炸板率>50%是警报，千亿涨停要兑现
- 不追求卖最高点，吃完鱼身就离场

### 冬藏
- 管住手≤10%，量化集体卖出要空仓
- 多杀多凶险，Tushare监控救性命
- 无利空闪崩是量化踩踏，绝对空仓不抄底
- 等待量化转向日，数据确认再春播
