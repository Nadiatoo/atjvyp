# 庄稼人战法 · 实战分析方法论

> 基于情绪周期的四季轮回框架，提炼可复用的市场分析 checklist

---

## 一、每日市场分析三步法

### Step 1：判断当前季节（情绪定位）

| 季节 | 判断标准 | 操作策略 |
|------|----------|----------|
| **春播** | 近期强势股补跌完成+恐慌盘杀出 | 开始建仓，分仓布局 |
| **夏长** | 主线板块与大盘共振向上 | 持股待涨，挖掘补涨 |
| **秋收** | 板块全面高潮+大块头涨停 | 逐步兑现，去弱留强 |
| **冬藏** | 高位股杀跌+题材半日游 | 空仓观望，喝茶看景 |

**关键问题：**
- 最近逆势逞强的板块/个股补跌了吗？
- 今天涨停家数、连板高度、炸板率如何？
- 昨日涨停股今天溢价如何？

---

### Step 2：识别主线板块（三要素检查）

一个能成为主线的板块必须同时满足：

**☑️ 天时（情绪）**
- 出现在春播期或夏长初期
- 能带动市场情绪共振向上

**☑️ 地利（位置）**
- 板块指数在均线附近或低位
- 近期无大量获利盘套牢盘

**☑️ 人和（逻辑）**
- 有清晰的产业预期/政策催化
- 相比其他板块逻辑鹤立鸡群
- 潜在多头多（看懂的人还少）

**实操 checklist：**
- [ ] 板块近20日涨幅排名？
- [ ] 板块内个股平均位置？
- [ ] 是否有硬核产业逻辑支撑？
- [ ] 能否带动大盘/相关板块跟涨？

---

### Step 3：制定交易计划

**春播期计划模板：**
```
恐慌信号：________（强势股补跌/跌停家数增加/大盘放量杀跌）
候选板块：________（符合三要素的1-3个板块）
分仓策略：每个板块配置____%仓位，共____只标的
止损位：板块跌破____日线/个股跌破____%
预期持有时长：____天
```

**夏长期计划模板：**
```
主线确认：________（已引领大盘共振）
持仓策略：核心股持有+挖掘补涨分支
补涨方向：________（低位的____属性股）
秋收信号：板块指数大涨____%+/大块头涨停/全面高潮
```

**秋收期计划模板：**
```
兑现节奏：
- 第一档（高潮日）：卖出跟风分支，减仓____%
- 第二档（反包后）：卖出中后排，保留核心龙头
- 第三档（次反包后）：彻底清仓

底线纪律：跌破____日线无条件清仓
```

## 二、四季转换识别信号

### 冬藏 → 春播（关键转折点）
**必须同时满足：**
1. 近期所有逆势逞强的板块都完成了补跌
2. 市场出现恐慌性杀跌（跌停家数激增）
3. 次日/当日有板块站出来引领反弹

**易错点：**
- 冬藏期会有多次假反弹，只有强势股补跌后的才是真春播
- 不要见到恐慌就开仓，要等补跌流程走完

### 春播 → 夏长
**确认信号：**
- 主线板块连续2-3日领涨
- 大盘与板块共振向上
- 出现明显的赚钱效应扩散

### 夏长 → 秋收
**核心信号（高潮一致）：**
- 板块指数单日大涨7%+
- 板块内大块头涨停或冲击涨停
- 跟风分支鸡犬升天，甚至无关个股也被炒作
- 连板高度达到近期峰值

**注意：** 秋收是一个阶段，不是一天

### 秋收 → 冬藏
**确认信号：**
- 主线板块反包技能卡用完（通常1-2次反包后）
- 板块核心股见顶回落
- 题材持续性变差（半日游）
- 涨停家数锐减，炸板率上升

## 三、不同市道的周期特征

### 大盘上涨趋势中
- **冬藏期**：很短（1-2天），快速洗牌
- **夏长期**：很长，多轮情绪周期叠加
- **策略**：重仓位参与，每一次春播都是机会

### 大盘震荡市中
- **四季时长**：相对均衡
- **策略**：控制仓位，快进快出
- **关键**：选对主线比择时更重要

### 大盘下跌趋势中
- **冬藏期**：很长，多次冰点
- **夏长期**：很短，闪电战
- **策略**：降低预期，小仓位博弈，见好就收
- **纪律**：等强势股补跌+多方持续性确认后再入场

## 五、心法口诀

**春播：**
- 无恐慌，不开仓
- 强势股补跌才是真春播
- 分仓布局，不怕倒春寒

**夏长：**
- 持股待涨，让利润奔跑
- 补涨看位置，低位的才安全
- 高潮时舍得卖

**秋收：**
- 去弱留强，逐步兑现
- 反包技能卡用完后彻底离场
- 没有完美的交易，赚了就跑

**冬藏：**
- 管住手，脉冲是诱饵
- 观察强势股补跌，等待春播
- 复利的前提是回避大回撤

## 六、常见错误自查表

| 错误 | 后果 | 修正方法 |
|------|------|----------|
| 冬藏期抢反弹 | 被诱多套牢 | 等强势股补跌后再入场 |
| 春播期一把梭 | 遇倒春寒被动 | 分仓布局，补票机会 |
| 夏长期频繁换股 | 错失主升浪 | 认定主线就耐心持有 |
| 秋收期舍不得卖 | 利润回撤 | 高潮一致逐步兑现 |
| 错过春播追高 | 高位站岗 | 等下一轮周期 |

---

**核心认知：**
1. 投机如山岳般古老，情绪周期永远轮回
2. 善猎者必善等待，善战者无赫赫之功
3. 复利=高胜率+低回撤，不是单次暴利
4. 市场由人组成，人性贪嗔痴永不变
